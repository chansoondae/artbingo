# 아트프렌즈 2025 전시 빙고 게임 - 기술 명세서

## 프로젝트 개요

아트프렌즈 커뮤니티 회원들이 2025년에 다녀온 미술 전시를 기록하고 공유할 수 있는 인터랙티브 빙고 게임 웹앱

### 핵심 컨셉
- "올해 미술 전시 어디까지 가봤니?"
- 전시 방문 기록을 빙고 형식으로 재미있게 추적
- 완성된 빙고판을 이미지로 저장하여 네이버카페에 인증

---

## 빙고 칸 상세 스펙

### 칸 구성 (16칸 = 4×4)
- **고정 칸**: 12칸 (전시 목록 제공 예정)
- **빈칸**: 4칸 (사용자 직접 입력)

### 각 칸의 데이터 구조
```typescript
interface BingoCell {
  id: string;
  type: 'preset' | 'custom';      // 고정 전시 or 사용자 입력
  museum: string;                  // 미술관명 (예: "국립중앙")
  exhibition: string;              // 전시명 (예: "내셔널갤러리")
  backgroundImage?: string;        // 흐릿한 배경 이미지 URL
  posterImage?: string;            // 전시 포스터 이미지 URL
  isVisited: boolean;              // 방문 여부 (도장 상태)
  order: number;                   // 드래그앤드롭 순서
}
```

### 칸 디자인 상세

#### 기본 상태 (미방문)
```
┌─────────────────┐
│  [배경이미지]    │  ← 흐릿한 배경 (opacity: 0.3)
│                 │
│    미술관명      │  ← 작은 글씨 (text-sm)
│    전시제목      │  ← 큰 글씨 (text-lg, font-bold)
│                 │
└─────────────────┘
```

#### 방문 완료 상태
```
┌─────────────────┐
│  [배경이미지]    │
│                 │
│    미술관명      │
│    전시제목      │
│   🔴 도장       │  ← "다녀옴" 또는 체크 스탬프
│                 │
└─────────────────┘
```

#### 빈칸 상태 (입력 전)
```
┌─────────────────┐
│                 │
│    + 추가하기    │  ← 클릭하면 입력 모달
│                 │
└─────────────────┘
```

## 기능 명세

### 1. 방문 체크 기능
- **동작**: 칸 탭/클릭 시 방문 상태 토글
- **시각적 피드백**: 
  - 도장 이미지 표시 (예: 빨간 원형 도장, "다녀옴" 텍스트)
  - 또는 체크마크 오버레이
- **애니메이션**: 도장 찍히는 효과 (scale + fade-in)

### 2. 드래그 앤 드롭
- **라이브러리**: @dnd-kit/core (React 18+ 호환)
- **동작**: 칸 위치 교환 (swap)
- **모바일**: 터치 드래그 지원
- **시각적 피드백**: 드래그 중인 칸 하이라이트

### 3. 빈칸 사용자 입력
- **트리거**: 빈칸 클릭/탭
- **입력 UI**: 모달 또는 인라인 입력
- **입력 필드**:
  - 미술관명 (필수)
  - 전시제목 (필수)
  - 포스터 이미지 (선택, 파일 업로드)
- **저장**: localStorage에 저장하여 새로고침 후에도 유지

### 4. 이미지 저장 기능
- **라이브러리**: html-to-image 또는 html2canvas
- **출력 형식**: PNG
- **저장 영역**: 빙고판 전체 (로고 ~ 하단 문구 포함)
- **파일명**: `아트프렌즈_전시빙고_2025.png`

### 5. 빙고 달성 표시 (선택적 기능)
- 가로, 세로, 대각선 4칸 완성 시 "빙고!" 표시
- 전체 완료 시 축하 메시지

## 필요한 외부 리소스

### 사용자 제공 필요
1. **아트프렌즈 로고** (PNG/SVG)
2. **2025년 전시 목록** (12개)
   - 미술관명
   - 전시제목
   - (선택) 배경 이미지
   - (선택) 포스터 이미지

### 자체 제작 필요
1. **도장 이미지**: "다녀옴" 텍스트가 들어간 빨간 도장
2. **기본 배경 패턴**: 전시 포스터 없을 때 사용할 기본 배경
